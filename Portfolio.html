My Services

Web Design

I create beautiful, responsive websites that look great on all devices and deliver a seamless user experience. My designs combine visual appeal with intuitive navigation, ensuring every site is both engaging and easy to use. I focus on clean layouts, modern aesthetics, and functional design that meets both user needs and business goals.

Data Analysis

I work with data analysis using tools like SQL, Pandas, and other programming languages. With SQL, I query and manage structured data in databases. Pandas helps me manipulate and analyze datasets in Python, especially for tasks like cleaning, filtering, and aggregating data. I also use other languages like R or Python libraries such as NumPy and Matplotlib for deeper analysis and visualization. These tools help me turn raw data into useful insights efficiently.

Web Development

I'm a web developer with a strong focus on frontend. I build responsive, user-friendly interfaces using HTML, CSS, JavaScript, and frameworks like React. I also have basic backend knowledge—enough to handle APIs, databases, and simple server setups using tools like Node.js or Express. I enjoy creating smooth, clean web experiences from design to deployment.

Video Editor / Photography

I'm a video editor and photographer with a passion for visual storytelling. I shoot, edit, and create content that captures emotion and detail. From framing the perfect shot to crafting smooth edits, I focus on bringing ideas to life through visuals.

Mobile App Design

I design user-friendly mobile applications that provide an exceptional user experience. My designs are tailored to meet the specific needs of your target audience while maintaining consistency with your brand identity.

Branding & Identity

I create cohesive brand identities that help businesses establish a strong presence in the market. From logo design to brand guidelines, I ensure that your brand communicates your values and resonates with your audience.
